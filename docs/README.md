# NGO Security Monitoring Documentation (GitHub Pages)

This documentation is built to be hosted via GitHub Pages (docs/).

## Overview

Follow the README in the repository root for installation and usage.

## Architecture

- A single Linux server runs scanner scripts via cron.
- Reports are saved under `/var/ngosec/reports`.
- Optional Node.js dashboard serves reports on port 3000.

## Getting Help

If you need help, open an issue in your GitHub repo or contact the maintainer.
